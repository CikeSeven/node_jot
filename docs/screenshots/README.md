# Screenshots Directory

Put your app screenshots in this folder using the exact file names below:

- `docs/screenshots/notes-list.png` - Notes list page
- `docs/screenshots/note-editor.png` - Note editor page
- `docs/screenshots/devices-page.png` - Devices page
- `docs/screenshots/settings-page.png` - Settings page

Optional extra screenshots:

- `docs/screenshots/search-mode.png` - Search animation/page
- `docs/screenshots/archive-page.png` - Archive page
- `docs/screenshots/dark-mode.png` - Dark mode sample

Recommended image size:

- Desktop capture: 1600x900 or 1920x1080
- Mobile capture: 1080x2400 (or device native ratio)
